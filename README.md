
# My Anime by Miraz

A next-generation, immersive anime discovery platform featuring a neon-cyberpunk aesthetic, real-time tracking, and AI-powered insights.

## Features

*   **Neon Cyberpunk UI**: A highly polished, immersive interface with glowing accents and smooth animations.
*   **Real-time Tracking**: Manage your Watching, Completed, and Planning lists with instant updates.
*   **AI-Powered Enrichment**: Uses Google Gemini to fetch live details, summaries, and generate visual prompts for genres.
*   **Lazy Persistence**: Data is saved locally to your browser, ensuring you never lose your progress on reload.
*   **Character Chat**: Interactive AI companions (Luffy, Naruto, Gojo, etc.) powered by Gemini.

## Setup Instructions

1.  **Clone the repository**
2.  **Install dependencies**: `npm install`
3.  **Set up Environment Variables**:
    *   Create a `.env` file in the root.
    *   Add your Google Gemini API Key: `API_KEY=your_google_api_key_here`
4.  **Run Locally**: `npm run dev`

## Deployment

This project is optimized for **Vercel**.
1.  Import this repository to Vercel.
2.  Add the `API_KEY` in the Vercel Project Settings > Environment Variables.
3.  Deploy.
